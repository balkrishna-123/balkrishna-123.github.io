Full Stack Development

Definition: Full Stack Development involves building both the frontend (client-side) and backend (server-side) of a web application, including the database and deployment.

Frontend Development

The frontend is what users see and interact with. It includes:

  • HTML: Structures the content
  • CSS: Styles the content
  • JavaScript: Adds interactivity
  • Frameworks: React, Angular, Vue.js

Backend Development

The backend handles server-side operations, including:

  • Server: Node.js, Express, Python, Django, Ruby on Rails
  • Database: MySQL, MongoDB, PostgreSQL
  • APIs: REST, GraphQL
  • Authentication: JWT, OAuth

Development Tools & Practices

  • Version Control: Git, GitHub
  • Package Managers: npm, yarn
  • Deployment: AWS, Heroku, Netlify
  • DevOps: CI/CD pipelines, Docker

Example: Node.js + Express Server

// server.js
const express = require('express');
const app = express();
const port = 3000;

// Middleware to parse JSON
app.use(express.json());

// Basic route
app.get('/', (req, res) => {
  res.send('Hello, Full Stack Development!');
});

// Route with parameter
app.get('/user/:id', (req, res) => {
  const userId = req.params.id;
  res.send(`User ID: ${userId}`);
});

// POST route
app.post('/user', (req, res) => {
  const userData = req.body;
  // Save user to database
  res.status(201).json({ message: 'User created', data: userData });
});

app.listen(port, () => {
  console.log(`Server running at http://localhost:${port}`);
});

Full Stack Architecture

A typical full stack application follows the MVC (Model-View-Controller) pattern:

  • Model: Manages data and business logic
  • View: Displays data (UI components)
  • Controller: Handles user input and updates model/view

HTML (HyperText Markup Language)

Definition: HTML is the standard markup language for creating web pages and web applications. It structures the content of web pages.

Basic HTML Structure

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document Title</title>
</head>
<body>
  <h1>Main Heading</h1>
  <p>This is a paragraph.</p>
</body>
</html>

Common HTML Elements

  • Headings: <h1> to <h6>
  • Paragraphs: <p>
  • Links: <a href="url">Link Text</a>
  • Images: <img src="image.jpg" alt="Description">
  • Lists: <ul>, <ol>, <li>
  • Forms: <form>, <input>, <button>
  • Divisions: <div> (block container)
  • Spans: <span> (inline container)

HTML5 Semantic Elements

Semantic elements clearly describe their meaning to both browser and developer:

  • <header> - Introductory content
  • <nav> - Navigation links
  • <main> - Main content
  • <section> - Thematic grouping
  • <article> - Self-contained composition
  • <aside> - Sidebar content
  • <footer> - Footer content

CSS (Cascading Style Sheets)

Definition: CSS is used to style and layout web pages — to alter the font, color, size, and spacing of your content, split it into multiple columns, or add animations and other decorative features.

CSS Syntax

selector {
  property: value;
  another-property: value;
}

CSS Selectors

  • Element selector: p { color: blue; }
  • Class selector: .className { font-size: 16px; }
  • ID selector: #idName { background: #fff; }
  • Attribute selector: a[target="_blank"] { color: purple; }
  • Pseudo-classes: a:hover { text-decoration: underline; }

Box Model

Every element is a rectangular box with these properties:

  • Content: The actual content of the box
  • Padding: Clears an area around the content
  • Border: A border that goes around the padding
  • Margin: Clears an area outside the border

CSS Positioning

  • Static: Default positioning
  • Relative: Positioned relative to its normal position
  • Absolute: Positioned relative to its first positioned ancestor
  • Fixed: Positioned relative to the browser window
  • Sticky: Toggles between relative and fixed based on scroll position

Flexbox Example

.container {
  display: flex;
  justify-content: center; /* Horizontal alignment */
  align-items: center;     /* Vertical alignment */
  flex-wrap: wrap;         /* Allow wrapping */
}

.item {
  flex: 1;                 /* Grow and shrink equally */
  margin: 10px;
}

CSS Grid Example

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* Three equal columns */
  grid-gap: 20px;                     /* Space between grid items */
}

.item {
  grid-column: span 2; /* Item spans two columns */
}

Responsive Design with Media Queries

/* Default styles for mobile */
.container { padding: 10px; }

/* Styles for tablets */
@media (min-width: 768px) {
  .container { padding: 20px; }
}

/* Styles for desktops */
@media (min-width: 1024px) {
  .container { padding: 40px; }
}

Bootstrap Framework

Definition: Bootstrap is a free, open-source CSS framework directed at responsive, mobile-first front-end web development. It contains CSS and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

Key Features

  • Responsive grid system
  • Extensive prebuilt components
  • Powerful plugins built on jQuery
  • Mobile-first approach
  • Customizable with SASS variables

Bootstrap Grid System

JavaScript

Definition: JavaScript is a programming language that enables interactive web pages. It is an essential part of web applications and is supported by all modern web browsers.

JavaScript Basics

Variables
// Variable declarations
var oldVariable = "I'm the old way"; // Function scoped
let changeableVariable = "I can change"; // Block scoped
const constantVariable = "I cannot change"; // Block scoped, constant
Data Types
// Primitive types
let string = "Hello World";      // String
let number = 42;                 // Number
let boolean = true;              // Boolean
let nothing = null;              // Null
let notDefined = undefined;      // Undefined
let symbol = Symbol('symbol');   // Symbol (ES6)

// Reference types
let array = [1, 2, 3];           // Array
let object = {                   // Object
  name: "John",
  age: 30
};
Functions
// Function declaration
function greet(name) {
  return `Hello, ${name}!`;
}

// Function expression
const greet = function(name) {
  return `Hello, ${name}!`;
};

// Arrow function (ES6)
const greet = (name) => {
  return `Hello, ${name}!`;
};

// Shorter arrow function
const greet = name => `Hello, ${name}!`;
Conditional Statements
// If statement
if (condition) {
  // code to execute if condition is true
} else if (anotherCondition) {
  // code to execute if anotherCondition is true
} else {
  // code to execute if all conditions are false
}

// Switch statement
switch(expression) {
  case value1:
    // code to execute when expression equals value1
    break;
  case value2:
    // code to execute when expression equals value2
    break;
  default:
    // code to execute when no cases match
}
Loops
// For loop
for (let i = 0; i < 5; i++) {
  console.log(i);
}

// While loop
let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

// For...of loop (for arrays)
const arr = [1, 2, 3];
for (const element of arr) {
  console.log(element);
}

// For...in loop (for objects)
const obj = { a: 1, b: 2, c: 3 };
for (const key in obj) {
  console.log(key, obj[key]);
}

DOM Manipulation

The Document Object Model (DOM) is a programming interface for HTML and XML documents.

Selecting Elements
// Select by ID
const element = document.getElementById('myId');

// Select by class name
const elements = document.getElementsByClassName('myClass');

// Select by tag name
const elements = document.getElementsByTagName('div');

// Query selector (returns first match)
const element = document.querySelector('.myClass');

// Query selector all (returns all matches)
const elements = document.querySelectorAll('.myClass');
Modifying Elements
// Change content
element.textContent = 'New text content';
element.innerHTML = '<strong>New HTML content</strong>';

// Change attributes
element.setAttribute('src', 'new-image.jpg');
element.classList.add('new-class');
element.classList.remove('old-class');
element.style.color = 'blue';

// Create new elements
const newElement = document.createElement('div');
newElement.textContent = 'I am a new div';
document.body.appendChild(newElement);
Event Handling
// Inline event handler
<button onclick="handleClick()">Click me</button>

// Using addEventListener
const button = document.querySelector('button');
button.addEventListener('click', function(event) {
  console.log('Button clicked!');
});

// Common events
element.addEventListener('click', handleClick);
element.addEventListener('mouseover', handleMouseOver);
element.addEventListener('mouseout', handleMouseOut);
element.addEventListener('keydown', handleKeyDown);
element.addEventListener('submit', handleSubmit);

ES6+ Features

  • Let and Const: Block-scoped variable declarations
  • Arrow Functions: Shorter function syntax
  • Template Literals: String interpolation with backticks
  • Destructuring: Extract values from arrays or objects
  • Spread/Rest Operator: ... for expanding or collecting elements
  • Promises: Handle asynchronous operations
  • Async/Await: Write asynchronous code that looks synchronous
  • Modules: Import/export functionality between files
